🎮 GameVerse — Modern Video Game E-Commerce Store
A sleek and immersive e-commerce web app for discovering and buying the latest video games.
Built with React, Tailwind CSS, and TypeScript for a smooth, responsive, and modern shopping experience.

🚀 Live Demo
🌐 Visit GameVerse — Coming soon!

✨ Features
🕹️ Hero Section with full-screen design and gradient highlights
💫 Smooth scroll animations for navigation
🎨 Dynamic lighting effects using Tailwind utilities
🛍️ Product listing section with game previews
🔍 Search & filter system (optional enhancement)
📱 Fully responsive for all devices
⚡ Fast performance powered by Vite
🧩 Tech Stack
Technology	Purpose
React	Frontend UI framework
Vite	Lightning-fast development environment
TypeScript	Type safety and maintainability
Tailwind CSS	Modern styling framework
shadcn/ui	Prebuilt accessible components
🛠️ Installation & Setup
Make sure you have Node.js and npm installed.

# 1️⃣ Clone the repository
git clone https://github.com/<your-username>/gameverse.git

# 2️⃣ Navigate to the project folder
cd gameverse

# 3️⃣ Install dependencies
npm install

# 4️⃣ Start the development server
npm run dev
